# Thumbsup

### What is this?

* This is a resource which adds the ability to thumbsup.

### Features
- To thumbsup press **`Del`**.

## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/helmimarif/Thumbsup
```

### Manually
- Download from this.

## Installation
- Add this to your `server.cfg`:

```
start Thumbsup
```
